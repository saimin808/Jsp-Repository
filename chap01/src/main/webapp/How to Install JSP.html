1. Eclipse Enterprise Edition 설치

  • 뒤에 for Web Developers 붙은 것으로 설치
  • File - New 했을 때 Dynamic Web Project가 안보인다면 우측 상단의 Open Perspective에서 JavaEE를 선택

Java Edition은 자바만 쓸 수 있다.

2. Apache Tomcat Web Application Server 설치

  • Web Application Server (WAS)
  • 9.0으로 설치
  • 압축 해제 후 Eclipse의 Servers 탭에 등록할 것

3. 새 프로젝트 생성하기

  • File - New - Dynamic Web Project
  • Next 두 번 눌러서 Generate web.xmld 생성을 체크 후 프로젝트 생성

4. 해당 프로젝트에 접속하기

  • 등록한 서버를 우클릭 후 Add and Remove에 프로젝트를 등록
  • 웹 브라우저에 직접 주소를 입력해 접속할 수 있다. (http://localhost:9000/프로젝트명/webapp내부경로)
  • 포트번호 변경은 등록한 서버의 server.xml을 수정해야 한다.
  • ※ WEB-INF폴더 밑에 만든 페이지로는 직접 접속이 불가능하다.
